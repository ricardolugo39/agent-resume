# Resume Agent

AI-powered job application manager that helps candidates generate tailored resumes, create cover letters, and track applications throughout the hiring process.


## Features

- AI resume generation
- AI cover letter generation
- Job application tracker
- SQLite database
- Streamlit interface
- Resume version history

## Tech Stack

Python
Streamlit
SQLite
OpenAI API
